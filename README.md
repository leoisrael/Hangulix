# Hangulix

Aplicação Node.js com renderização EJS e estética cyberpunk para treinar os caracteres do coreano.

## Como rodar
1. Instale o Node.js (>=18).
2. No diretório do projeto, execute:
   ```bash
   npm start
   ```
3. Acesse em `http://localhost:3000`.

## O que você encontra
- **Flashcards** para revelar a romanização e a composição de sílabas.
- **Quiz relâmpago** com feedback imediato para reforçar os jamos.
- **Tabelas visuais** com consoantes simples, duplas, vogais e ditongos.

Tudo embalado em laranja e preto, com uma vibe neon para deixar o estudo mais divertido.
